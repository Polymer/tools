## Workflow

### Setup
```bash
  npm install
  npm run bootstrap
  npm run build
```

### Development commands

```bash
  npm test # runs all command line tests, takes a couple of minutes

  code ./tools.code-workspace # opens vscode in workspace mode, configured for the monorepo
```
